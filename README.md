# Bellroy x Elm

This is a small coding exercise for the front end developer position at Bellroy.

## Getting Started

```bash
$ tar -xvzf bellroy.tar.gz bellroy
$ cd bellroy
$ open index.html
```

This will open a static HTML page with the Elm application properly loaded.

## Features

- [x] Navigate forward and backward using J and K
- [x] Turn left or right using H and L
- [x] Allow decreasing and increasing the size of the grid
